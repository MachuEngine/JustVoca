<기본 셋팅>
1. 토픽 단어 기준
 1) 오누이 적용 DB 활용
2. 목표량 설정
 1) 디폴트 10개? 추가로 더 설정 가능
3. 학습자/선생님/관리자 페이지
 1) 선생님 : 학습자 별 진도, 점수 통계, 컨텐츠 관리 등
 2) 관리자 : 시스템 설정, 로그 확인, API 설정 등
 
<주요 기능>
1. 플래시 단어 학습
 1) 그림, 음성, 발음, 녹음, 뜻, 예문, 연습 문제
 2) 단어와 예문에 대한 녹음 제공
2. 발음/유창성 평가
 1) 생성형 AI 코멘트
3. 테스트
 1) 목표량 채웠을 경우 테스트 진행
 2) 듣기, 말하기, 쓰기
4. 누적 학습
 1) 학습한 단어를 누적하여 리스트로 보여주고 단어 또는 뜻을 가려 학습한 단어 상기
5. 오답노트
 1) 틀린문제들 모아서 형식 그대로(문제 틀리면 오답노트에 자동저장)
6. 복습기능
 1) 발음/유창성 점수가 일정 점수 이하일 경우 복습 메뉴에서 다시 학습

<확장 기능>
1. 문서 출력
 1) 테스트지와 오답노트 출력 (토픽을 페이퍼로 치루므로 대비용)
2. 나만의 단어장 (즐겨찾기 기능)
 1) 학습자가 직접 단어 추가
 2) 엑셀 import 기능
 3) 길게 누를 경우 추가 기능
3. 알림 서비스
4. 반복 학습
 1) 횟수나 시간 설정하여 반복적으로 TTS 출력
    (언제 어디서나 길을 걸어가면서 들을 수 있게)
5. 챗봇 (오누이 자문)
 1) 음성, 문자, 음성+문자 옵션 지원
 2) 대화 수준 레벨별 (CFRF A1~C2 또는 토픽1~6급으로 설정)
 3) 수준별로 적합한 대화 주제 선택
 4) 챗봇이 학습자에게 먼저 질문을 제시하는 방식의 프롬프트 설계
6. TTS 화자 선택
7. 학습 루틴 만들기
 1) 일정한 시간과 요일 설정
 2) 그 시간에 접속하지 않으면 알림 울림
 3) 시간에 맞춰 분량이 자동 조절 (new 단어 + 랜덤 복습)
8. 싸이클 학습
 1) 단어별로 '발음 듣기 -> 빙고 -> 스펠링 듣고 적기 -> 스피드 퀴즈 -> 빈칸 채우기' (ex.보카트레인) 단계의 과정을 항상 거치면서 학습
9. 문법 설명
10. 게임요소
