student_home.html (학습하기 화면)
  > 첫번째 로그인 시(학생)에는 단계 선택 후 해당화면 출력
  > 두 번째 이후 로그인(학생)부터 해당 화면 바로 출력
  > 오늘의 학습 목료량(10개. 현재 목표량 설정 화면 없음)이 완료되면, 오늘 학습 부분에 대한 확인학습(3문항) 풀기
  > 앞장 : 레벨 선택 버튼, 급수, TOPIK 단계, 이미지, 단어, 한글 발음, 사전적 뜻, 뜻 영어 번역, 발음 듣기 구성
  > 뒷장 : 예문, 문장 듣기, 문장 녹음, 이전, 다음 버튼 구성
  > 예문 문장 녹음하여 결과 제시 : 전체 점수, 어절 점수, 음절 점수, 음소 점수 구성
  > 학습 내용 이어서 점검하는 형태의 문제들로 단어 10개 학습 시 3문제 구성
  > 학습 완료되면 결과에 따라, 다시 문제를 풀 지, 다음 단계를 조금 더 학습할지, 학습을 완료할 지 선택
     - 이어서 학습하기 >> 다음 10개 단어 학습 시작
     - 다시 풀기 >> 오늘 학습 점검한 3개 문항 다시 풀기
     - 오늘 학습 완료 >> study_complete.html 로 이동


study_complete.html (학습 종료 화면)
  > 오늘 학습 완료된 부분에 대한 칭찬 등 동기부여용 화면
  > 그 날의 학습에 대하여 AI가 한줄 피드백 제공
  > 학습 단어 개수, 몇번을 들었는지 등 간단하게 보여줌
  > 현재까지 진행 된 진도율 등에 대하여 가시적으로 보여주고 다음 학습 내용 안내
  > 홈으로를 누르면 student_home.html로 이동
        * 추후에는 오늘 학습 목표량 설정하는 화면으로 이동 예정
  > 복습하기를 누르면, review.html로 이동

review.html (복습하기 화면)
  > 시연용 html 미생성
  > 오늘 학습 점검 문제에서 틀린 문제 다시 풀 수 있도록 함

teacher_dashboard.html
  > 학습 통계에서 전체학생 수 / 진도 평균 / 과제제출 학생 수 / 시험 평균 점수를 확인할 수 있으며, 각각의 항목 클릭 시 하단에 학생 리스트와 정보가 출력되며, 특이사항이 있는 학생의 경우 붉은 글씨로 표기됨(과제 미제출/장기 미출석 등)

      진도 평균 - 현재 학습 현황
      과제제출 - 제출한 과제파일 및 제출일자 확인 가능
      시험 평균 - 맞고 틀린 항목 개수 및 AI 평가 결과
  > 특정 학생 검색 가능(검색 또는 엔터) 
  > 본인의 학생 계정 비밀번호를 초기화해줄 수 있음
  > 공지 발송 가능 (즉시/예약) 및 공지 발송 이력 남음
